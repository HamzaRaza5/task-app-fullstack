# Task App Full Stack

A full-stack task management web application built with React, NestJS, and PostgreSQL.

## Tech Stack

- React + Vite
- TypeScript
- NestJS
- PostgreSQL
- JWT Authentication

## Project Structure

task-app-fullstack/
├── react-frontend
└── task-app

## Features

- User signup
- User login
- JWT-based authentication
- Active task management
- Completed task management
- Protected routes

## Frontend Setup

cd react-frontend
npm install
npm run dev

## Backend Setup

cd task-app
npm install
npm run start:dev
